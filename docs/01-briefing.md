# Briefing do Projeto

## Tipo de Projeto
Aplicação Web / SaaS

## Objetivo de Negócio
Assistente financeiro inteligente que combina análise macroeconômica com carteira pessoal dos usuários, oferecendo consultoria de investimentos via WhatsApp

## Público-Alvo
Investidores ativos autodidatas, 28-45 anos, renda R$ 8-35 mil, patrimônio R$ 30-500 mil, profissionais liberais e empresários

## Nicho de Mercado
Fintech de inteligência artificial para consultoria de investimentos

## Escopo Desejado
Plataforma de inteligência financeira com análise macroeconômica, monitoramento de carteira, IA conversacional via WhatsApp e dashboard web. Inclui webscraping, sistema de watchlist, múltiplos planos de assinatura e compliance CVM

## Stack Tecnológica
Frontend: Lovable (React), Backend/DB: Supabase, Automação: n8n, IA: Claude API, Pesquisa: Perplexity API, WhatsApp: Evolution API, Dados: B3/Yahoo/Alpha Vantage, Pagamentos: Stripe/Asaas

## Biblioteca de Componentes UI
shadcn/ui

## Prazo
2 meses

## Orçamento
baixo

## Critérios de Sucesso
Posicionamento entre robo-advisors e assessorias tradicionais, transmitindo clareza, proximidade, precisão e confiança

## Restrições Técnicas
Desenvolvimento com Windsurf, desenvolvimento solo

## Compliance
Fase 1: Plataforma informativa/educacional sem recomendações personalizadas. Fase 2: Parceria com consultoria registrada CVM para recomendações no plano Advisory
